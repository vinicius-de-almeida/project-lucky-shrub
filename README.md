## About The Project

This project is a responsive website for "Lucky Shrub," a fictional garden design company. The primary goal was to build a clean, professional, and fully responsive online presence using modern web development best practices. The site features a semantic HTML5 structure and is styled with SCSS for maintainability and advanced features. The project was guided by Meta.
